# Whack A Mole

A Simple fun game, developed using HTML, CSS, JavaScript

## Usage
```
git clone https://github.com/jeyk333/whack-a-mole.git
cd whack-a-mole
//start the server
```

## JavaScript Methods and Functions used
- querySelector()
- querySelectorAll()
- textContent()
- forEach()
- classList.remove()
- classList.add()
- Math.floor()
- Math.random()
- addEventListener()
- setInterval()
- clearInterval()
- alert()
